# vim快捷命令

## 以下命令均在命令模式下操作

1. 转到文件结尾：

    ```bash
    G
    ```

2. 转到第9行：

    ```bash
    9G
    ```

3. 删除所有内容(先用G转到文件末尾)：

    ```bash
    :1,d
    ```

4. 删除第9行到第200行的内容(先用200G转到第200行):

    ```bash
    :9,d
    ```

5. 删除光标所在处字符:

    ```bash
    x
    ```

6. 删除光标所在前字符：

    ```bash
    X
    ```

7. 删除到下一个单词开头:

    ```bash
    dw
    ```

8. 删除到本单词末尾:

    ```bash
    de
    ```

9. 删除到本单词末尾包括标点在内:

    ```bash
    dE
    ```

10. 删除到前一个单词:

    ```bash
    db
    ```

11. 删除到前一个单词包括标点在内:

    ```bash
    dB
    ```

12. 删除一整行:

    ```bash
    dd
    ```

13. 删除光标位置到本行结尾:

    ```bash
    D 或 d$
    ```

14. 删除光标位置到本行开头:

    ```bash
    d0
    ```
